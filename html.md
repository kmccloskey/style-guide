# HTML5: Style Guide
## New York Code + Design Academy

### Indentation

Every new block level element should be indented.

Good

```html
<div class="wrapper">
  <ul>
    <li>
      <a href="http://google.com">Google</a>
    </li>
  </ul>
</div>
```

Bad

```html
<div class="wrapper"><ul>
    <li>
      <a href="http://google.com">Google</a></li>
</ul></div>
```
